uint8_t CDC_Transmit_FS_Blocking ( uint8_t * Buf,
uint16_t Len,
uint32_t timeout_ms )

Отправляет данные по USB CDC с ожиданием готовности передатчика (retry-цикл).

Note
Оборачивает CDC_Transmit_FS, повторяя попытку до истечения timeout_ms, если хост временно не готов принять пакет (передатчик занят, USBD_BUSY).
Parameters
BufУказатель на буфер данных для отправки.
LenДлина данных, байт.
timeout_msМаксимальное время ожидания готовности передатчика, мс.
Return values
USBD_OKпри успешной отправке; USBD_FAIL при истечении времени ожидания.
